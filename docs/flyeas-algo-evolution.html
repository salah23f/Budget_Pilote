Detail des ameliorations

V2 — Pre-seed historique

+1.9%
Au lieu d'attendre 5 samples, on pre-charge 60-180 fares historiques via Tequila API a la creation de mission.
Coût : €30/mois · ROI : excellent

V3 — Mistake fare detector

+1.6%
Module parallèle qui detecte les anomalies (prix < mean - 3σ + drop volume) et trigger instant-buy.
Coût : €50/mois · ROI : bon (P95 double)

V4 — ML model

+1.7%
Remplace scoring additif par gradient boosting entraîne sur 2M trajectoires.
Coût : €200/mois · ROI : moyen, complexe

V5 — Multi-airport flex

+2.4%
Scan aéroports voisins + ±2 jours de dates. Gros gain, mais 3-5x plus d'appels API.
Coût : €500/mois · ROI : excellent

V6 — Regional models

+1.5%
Sub-modèles par type de route (court/long, weekend/business, saisonnier).
Coût : €800/mois · ROI : moyen

Total V1 → V6

+9.1%
Savings presque doublent (11.3% → 20.4%). Win rate de 68% → 87%.
Coût total : €1,580/mois · Impact: transformationnel
